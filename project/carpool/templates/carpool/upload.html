Instructions:

  • File Format: .csv
  • Fields Order and Description:
    • id: student's id, must be unique
    • zip_code: 5 digits zip code of student's residence
    • role: D, E, or P - for Driver, Either, or Passenger
    • seats: number of available seats in the vehicle
    • gender: F, M, or N - for Female, Male, or Non-binary
    • under_25: 0 for False, 1 for True - Is student age group under 25 years?
    • undergrad: 0 for False, 1 for True - Is student academic level undergraduate?
    • smoker: 0 for False, 1 for True - Is student a smoker?

    • pref_age: 0 for False, 1 for True - Does the student prefer to ride with people of the same age group?
    • pref_status: 0 for False, 1 for True - Does the student prefer to ride with people of the same academic level group?
    • pref_nonsmoker: 0 for False, 1 for True - Does the student prefer to ride with non-smoker people?

    • T_arr: Tuesday time of arrival in campus using 24h format from 7:00 to 22:00
    • T_dep: Tuesday departure time from campus using 24h format from 7:00 to 22:00

    • W_dep: Wednesday departure time from campus using 24h format from 7:00 to 22:00
    • R_arr: Thursday time of arrival in campus using 24h format from 7:00 to 22:00
    • R_dep: Thursday departure time from campus using 24h format from 7:00 to 22:00
    • F_arr: Friday time of arrival in campus using 24h format from 7:00 to 22:00
    • F_dep: Friday departure time from campus using 24h format from 7:00 to 22:00
